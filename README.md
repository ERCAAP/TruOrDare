# Truth or Dare Game | Doğruluk mu Cesaret mi?

A modern and engaging mobile application that brings the classic Truth or Dare game to life with animated stickers, multiplayer support, and premium features.

Modern ve ilgi çekici bir mobil uygulama olan klasik Doğruluk mu Cesaret mi oyununu animasyonlu çıkartmalar, çok oyunculu destek ve premium özelliklerle hayata geçiriyor.
